# Amino-Acid-Validity-
#Beginner
BIOLOGICAL SIGNIFICANCE:
"The code works to determin the Invalid Amino Acids present in the protein Sequence. 
Amino Acids make up proteins and the valid amino acids in the sequence are esential to work with proteins.
Invalid amino acids don't make up proteins the invalid amino acids decrease the protein stability"
WORKING OF CODE
This code take the input from user in the from of a string and make it Upper case and the Amino Acids are in Upper case letters and remove the white spaces form the sequence. 
The flag variable is made for correct sequence.
The new variable is made in order to correct the protein sequence by removing the invalid sequences.
The seuqence is then compaired with the valid amino acid sequnce containing the 20 amino acids. 
If any amino acid not present in the valid sequence it specifies it using the flag variable. 
The determined invalid amino acids along with their positions are displayes on the terminal. 
In case of invalid sequence the invalid amino acid are removed and correct amino acids are added in the varriable corrected_sequence. 
In case of no invalid amino acids "The protein Sequence is Valid" is printed.
